## Features

- Attractive UI
- Responsive design for various screen sizes
- Replay the match after winning
- Cool sound effects


## How to Play

1. Clone the repository:

    ```bash
    git clone https://github.com/amit0-git/Tic-Tac-Toe.git
    ```

2. Open `index.html` in your preferred web browser.

3. Enjoy the game!
